# Todoリストアプリ

Laravel学習用として作成しているTodoリストのWebアプリです。

## 開発経緯

AED地図案内アプリを作成している中で、未経験のLaravelに苦戦する場面が多く、一度しっかりとMVCの構造や基本的な操作などを学ぶ必要性を痛感しました。そこで、CRUDアプリの代表例であるTodoリストを作成し、Laravelの「基礎の基礎」を学ぶことにしました。

## 主な機能

一般ユーザと管理者の二つのロールを用意します。各ロールの機能は、以下に示すものを予定しています。

### 一般ユーザ：

- ユーザログイン・ログアウト（✓済み）
- ユーザ登録・退会（✓済み）
- Todoリストの登録・編集・削除（✓済み）
- 登録済みTodo参照（✓済み）

### 管理者：

- 管理者ログイン
- ユーザ一覧の参照
- ユーザログ参照
